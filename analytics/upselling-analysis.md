## 1. 현재 감정 아크 진단

프레젠테이션은 **Rebirth 아크** (각성 → 변화)를 따름:

```
stagnation → awareness → pain → growth → renewal
(좋은 사람으로 끝남) → (원리가 있다) → (8년 배신) → (메커니즘/칼리브레이션) → (정원을 가꿔라)
```

**업셀링 진입에서 아크가 깨짐:**

```
renewal(정원을 가꿔라) → 갑자기 pain(너 혼자 못 해) → CTA
         ↑ 감정 최고점                ↑ 급추락
```

Rebirth 아크의 **"resolution_type: transformative"가 파괴**됨. 청중이 정원 비유에서 "나도 할 수 있겠다"는 renewal 감정에 도달했는데, 갑자기 "못 해"로 끌어내리면 **인지 부조화** 발생.

## 4. 핵심 모순

> "네 정원을 가꿔라" (= 네 기준을 세워라, 주체적이어라)
> ↓
> "근데 혼자 못 해, 우리가 필요해" (= 너 부족해)

청중의 무의식적 저항: **"아까는 내가 기준을 세우라더니 이제 너네한테 의존하라고?"**

코칭의 포지션 문제:
- 기존: "너 약하니까 도와줄게" (구원자)
- 필요: **"네 정원의 설계도를 같이 그리는 사람"** (파트너)

## 5. 제안: 정원 비유 관통 구조

### 변경 전
```
정원을 가꿔라 (renewal)
    ↓
KNOWING≠DOING (Loss Aversion)
    ↓
66일 (Authority)
    ↓
관성의 법칙 (FOMO + Loss Aversion)
    ↓
코칭 소개 → CTA
```

### 변경 후
```
정원을 가꿔라 (renewal)
    ↓
"여러분의 정원은 어떤 모습입니까?" (Self-Reference #17 — 자기 참조)
    ↓
케이스 3개를 "정원 가꾸기 과정"으로 리프레이밍
    ↓
"혼자 씨앗을 심을 수는 있다. 근데 어떤 씨앗인지 아는가?" (Curiosity Gap #1)
    ↓
코칭 = 정원 설계사 (Authority #5 + Reciprocity #4)
    ↓
CTA
```

### 트리거 조합 변경

| 기존 | 변경 |
|---|---|
| Loss Aversion (공포) | **Curiosity Gap** (호기심) |
| "못 한다" (부정) | **Self-Reference** "네 정원은?" (질문) |
| FOMO | **Progress Illusion** #19 "이미 50% 왔다" |

### 예상 감정선

```
renewal → curiosity → hope → commitment → action
(정원 가꿔라) → (내 정원은?) → (이 사람들도 됐네) → (나도 되겠다) → (신청)
```

**공포 없이, 자연스러운 호기심과 희망으로 업셀링 연결.**

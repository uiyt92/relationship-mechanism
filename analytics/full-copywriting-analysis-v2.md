- **왜 강한가**: 프레젠테이션 전체의 중심 메타포를 시각적으로 완성
- 쫓는 사람(빨강/하락) vs 가꾸는 사람(초록/상승)의 Contrast #9가 애니메이션으로 즉시 체감

  - slide14에서 이미 "호기심 → 신뢰 → 가치 = 구매"를 보여줬으므로, slide15d는 **연애 맥락으로 치환**하는 것이 효과적

## 7. 업셀링 파트 전환 자연스러움 분석

### 정원 비유 → 업셀링 감정 아크 연속성

```
[교육 파트 절정]                    [전환 구간]                    [업셀링 파트]
cal-3: 정원 비유 (10/10)   →   slide20~video (실전 증명)   →   slide28~32 (멤버십)
  "정원을 가꿔라"                 "이렇게 하면 된다"              "설계도가 필요하다"
  감정: 깨달음+희망              감정: 확인+갈망                 감정: 필요+결심
```

#### 전환 자연스러움 평가: 8.5/10

**잘 된 점:**

1. **메타포 일관성** (10/10): "정원 비유"가 cal-3에서 시작되어 slide28-2("잘못된 씨앗 → 잡초밭 / 올바른 씨앗 → 정원"), slide30("씨앗부터 정원 완성까지"), slide30-proof("나비가 찾아오는 정원"), slide31("첫 번째 꽃이 피는 순간")까지 끊김 없이 이어짐. 이 메타포 일관성이 업셀링을 "판매"가 아닌 "다음 단계"로 느끼게 만드는 핵심 장치.

2. **KNOWING ≠ DOING 브릿지** (9/10): slide28에서 "아는 것 ≠ 되는 것"을 선언한 뒤, 66일 법칙(UCL 연구)으로 과학적 근거를 제시. 이것이 "혼자는 어렵다"는 메시지를 강압적이 아닌 논리적으로 전달.

3. **Self-Reference 연쇄** (9/10): katalk-bridge("칠 수 있습니까?") → slide28("씨앗의 종류를 아는 것과 심어서 꽃을 피우는 것") → slide31("테스트를 걸었을 때 흔들리지 않는 자신")으로 Self-Reference가 3연타. 청중이 자기 상황에 계속 대입하게 만듦.

**약한 점:**

1. **Social Proof 집중도** (7/10): slide30-community, slide30-proof, slide30-feedback, slide30-meetup이 연속 4장. Social Proof가 쌓이는 건 좋지만, 감정 아크가 "증명 나열" 모드로 전환되어 감정선이 평탄해짐.
   - **개선안**: 4장 중 1장을 줄이거나, 중간에 "여러분도 이 정원에서 함께할 수 있습니다" 같은 감정 CTA를 삽입

2. **가격 앵커링 전환** (7.5/10): slide31(감정 CTA)에서 slide32(가격)로의 전환이 약간 급함. "첫 번째 꽃이 피는 순간"이라는 감정적 절정 직후 바로 가격표가 나옴.
   - **개선안**: slide31과 slide32 사이에 "이 정원을 시작하는 데 필요한 것은 단 하나입니다" 같은 1장 브릿지 추가
